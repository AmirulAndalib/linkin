### Host in

### To develop locally 
#### Clone and install dependancies

```
git clone https://github.com/RizkyRajitha/linkin.git
cd linkin
npm i
npm run dev
```
#### Database migration

**you must have postgres database setup locally**

```
node scripts/migrate.js 
```
